Smart Helmet Detection
System for Motorcyclists

An IoT safety system that captures a motorcyclist image using Raspberry Pi, analyzes helmet usage on a local server, syncs the result with Firebase, and controls a physical traffic light and OLED display.

Explore System Flow View Results

Problem & Solution

The system addresses a real road-safety challenge through an automatic and scalable IoT solution.

The Problem

Helmet use is a major factor in motorcyclist safety. At intersections, enforcement is often manual or inconsistent, and a rider without a helmet can enter traffic without immediate automatic response.

Our Solution

The system detects a motorcycle rider and checks helmet usage. If a rider is detected without a helmet, the Raspberry Pi can keep the traffic light red and show a warning on the OLED display.

How It Works

A clear IoT pipeline: edge capture, local server processing, cloud logging, and physical hardware response.

STEP 01

Image Capture

The Raspberry Pi camera captures an image of the motorcyclist at the intersection.

STEP 02

Server Upload

The image is sent over the local network to the processing server.

STEP 03

Model Analysis

The computer vision model detects the motorcycle, rider, and helmet status.

STEP 04

Firebase Log

The detection result is saved to Firebase Firestore as structured JSON data.

STEP 05

Physical Output

The Raspberry Pi uses the result to control the traffic light and OLED warning display.

Detection Logic

Two-Stage Vision Analysis

The project uses two detection stages. The first detects people and motorcycles. The second checks helmet usage. The logic then matches the rider to the motorcycle and checks the rider head area for helmet detection.

  • Motorcycle detection: verifies that the scene contains a motorcycle.
  • Rider association: checks whether a detected person belongs to the motorcycle area.
  • Helmet validation: checks whether the rider has a detected helmet on the head region.

Computer Vision Result

The final output is a structured result containing motorcycle presence, rider count, helmet count, and whether green light is allowed.
